### 프로젝트 소개

구걸 마스터: 프리미엄 에디션은
클릭(구걸) → 강화 → 자산 투자 → 환생으로 이어지는
성장형 방치 RPG 게임입니다.

단순 클릭을 넘어
스킬 트리 / 자산 시스템 / 환생 시스템까지 포함된
## 풀 구조 클릭커 게임
### 주요 기능
# 1. 클릭 시스템
터치로 골드 획득
크리티컬 시스템 (최대 95%)
파티클 & 사운드 피드백
# 2. 강화 시스템
클릭 강화 (수동 수익 증가)
자동 수익 (초당 골드 생성)
다중 구매 (x10 / x100)
# 3. 스킬 트리
클릭 강화
자동 속도 증가
크리티컬 확률 증가
골드 보너스
자산 효율 증가
#4. 자산 시스템

## 4가지 카테고리:

부동산 → 클릭 강화
미술품 → 크리티컬 강화
기업 → 자동 수익 강화
도시 → 전체 배율 증가

자산 수집 = 영구 성장

# 5. 환생 시스템 (Prestige)
일정 골드 달성 시 환생 가능
모든 진행 초기화
대신 영구 배율 증가
# 6. 랜덤 이벤트
골드 2배
자동 수익 증가
크리티컬 강화
디버프 이벤트 포함
# 7. 업적 시스템
특정 조건 달성 시 보상
스킬 포인트 지급
# 8. 모바일 최적화
터치 기반 인터랙션
가로(탭) / 세로(컨텐츠) 스크롤 분리
관성 스크롤 지원 (iOS 대응)
### 기술 스택
HTML5 Canvas
Vanilla JavaScript
Web Audio API
LocalStorage (데이터 저장)
