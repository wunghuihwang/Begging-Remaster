💰 구걸 마스터: 프리미엄 에디션 (Begging Master: Premium Edition)
"단순한 구걸은 잊어라. 이제는 전략이다."
클릭에서 시작해 도시를 지배하는 자산가로 거듭나는 풀 구조 클릭커 방치형 RPG입니다.

🎮 게임 루프 (Core Loop)
단순 반복을 넘어선 유기적인 성장 시스템을 경험하세요.

클릭(Beg): 화면을 터치하여 기초 자산을 확보합니다.

강화(Upgrade): 클릭 효율과 자동 수익을 높입니다.

투자(Invest): 부동산, 미술품, 기업 등 자산을 수집하여 영구 버프를 획득합니다.

환생(Prestige): 성장이 정체될 때, 모든 것을 내려놓고 더 강력한 배율로 다시 시작합니다.

✨ 주요 기능
🖱️ 1. 하이퍼 클릭 시스템
크리티컬: 최대 95% 확률의 크리티컬 시스템으로 폭발적인 수익 창출.

피드백: Canvas 기반의 화려한 파티클 효과와 몰입감을 더하는 사운드 피드백.

📈 2. 심화된 강화 & 스킬 트리
다중 구매: x10, x100 단위 구매로 빠른 성장 지원.

전략적 스킬: 클릭 강화, 자동 속도, 크리티컬 확률, 자산 효율 등 커스텀 성장 지원.

🏢 3. 자산 시스템 (Asset Management)
단순한 숫자가 아닌, 테마별 자산 수집을 통한 영구적 능력치 상승.

부동산 → 클릭 강화

미술품 → 크리티컬 강화

기업 → 자동 수익 강화

도시 → 전체 배율 증가 (최종 테크)

♻️ 4. 환생 & 랜덤 이벤트
Prestige 시스템: 일정 골드 달성 후 환생 시 모든 진행도가 초기화되지만, 영구 배율 증가 버프 획득.

돌발 이벤트: 골드 2배, 자동 수익 증가 등 버프와 긴장감을 주는 디버프 이벤트 포함.

📱 5. 모바일 최적화 및 UX
하이브리드 스크롤: 가로(탭 전환)와 세로(컨텐츠) 스크롤 분리로 조작 편의성 극대화.

iOS 대응: 웹 환경에서도 부드러운 관성 스크롤 지원.

🛠 기술 스택 (Tech Stack)
Engine: HTML5 Canvas (고성능 그래픽 렌더링)

Logic: Vanilla JavaScript (프레임워크 없이 구현된 순수 로직)

Audio: Web Audio API (실시간 사운드 이펙트)

Persistence: LocalStorage (5초 주기 자동 저장 및 데이터 복구)
